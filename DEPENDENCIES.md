# External Tool Dependencies

This document lists the external tools and dependencies required by this dotfiles repository.

## Shell Environment

- **NVM (Node Version Manager)**: Used for managing Node.js versions. A `.nvmrc` file is included to specify the recommended version.
- **Cargo**: The Rust package manager. Required for Rust development.
- **Bun**: A JavaScript runtime and toolkit.
- **Starship**: A cross-shell prompt.

## Development Tools

- **Claude Code**: A custom tool located at `/home/coldaine/.claude/local/claude`.
- **GitHub CLI (gh)**: Used for Git credential management.
- **Gooey**: A custom-built tool. It is recommended to install this separately.
- **Genymotion**: An Android emulator.

## Terminal and Desktop Environment

- **Ghostty**: The terminal emulator.
- **KDE Plasma**: The desktop environment.
- **Polonium**: A KWin tiling extension.
- **Kate**: The text editor.
- **Dolphin**: The file manager.

## System Services

- **Sunshine**: A game streaming server.
